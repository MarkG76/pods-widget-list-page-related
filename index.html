Welcome!

This is a simple WordPress plugin related to the Pods Framework (v2.2). It will display multiple Pod items, related to the current page. The code is based on the equivalent "list items" widget of Pods. The interface is the same with an addition of adding an input to enter the name of the field which is linked to the page.

=Usage=

1) Create a new Pod and add a Relationship field linked to Pages 2) Go to Widgets and add the widget to an area. Set the "Page field" input to the name of the field that is related to the pages.
